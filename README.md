## Run Locally

Clone the project

```bash
  git clone https://github.com/MohitSaraff/Keeper.git
```

Go to the project directory

```bash
  cd Keeper
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
